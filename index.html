My Research

My Research

My research focuses on the interplay between Machine Learning and Programming Languages. In particular, I'm interested in reasoning about programming in a more ambiguous fashion without strict semantics, and reasoning about learning algorithms in a more structured way with more abstractions.

Projects

sk_p: correcting programs using neural networks

In this project we explore whether it is possible to correct a program by altering (via replacement, insertion, deletions) some statements so that the resulting program has fragments of codes that "look like" that of a correct program. To this end, we train a neural network model that is capable of generating candidate statements given its two surrounding statements, remembering the correlations between statements in a program fragment. We show this simple scheme is powerful in the context of MOOCs, where many students solve the same assignment problem, thus having a large corpus of shared code fragments. Our approach out-performs the state of the art method which uses symbolic techniques, and does not require manual correction models for each of the MOOCs assignments.

composible state abstractions in model learning

In this project we explore whether it is possible for an agent to automatically learn composible state abstractions. As one interact with their environment, they often makes useful abstractions to simplify the planning problem by planning on the abstract level. A key aspect of a good abstraction is that it is composible: One can seemlessly move between abstract states in a predictible way, without "falling outside" the abstraction during the proccess. We investigate if such property can be enforced during model learning, so that we learn a composible set of abstractions.
